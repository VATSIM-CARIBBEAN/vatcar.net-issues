Welcome to the **public issue tracker** for [vatcar.net](https://github.com/VATSIM-CARIBBEAN/vatcar.net) 
This repository is dedicated **only** to reporting bugs, requesting features, and discussing ideas.

> The main codebase for vatcar.net is private.  
> You can still contribute by helping us identify bugs and suggest improvements here!

---

## Report a Bug

If you find a bug, please [open a new issue](../../issues/new?template=bug_report.md) and include:

- A clear description of the problem  
- Steps to reproduce  
- Expected vs. actual behavior  
- Screenshots or logs (if applicable)  
- Your environment (browser, OS, device, etc.)

---

## Request a Feature

To propose a new feature or improvement, [open a feature request](../../issues/new?template=feature_request.md) and include:

- What problem it solves  
- Why it would be useful  
- Any ideas for implementation or UI

---

## Guidelines

- Be respectful and constructive.
- Avoid sharing sensitive or private data.
- Do not post source code from the private repository.

We review issues periodically and label them to indicate status (`bug`, `enhancement`, `discussion`, etc.).

Thank you for helping improve **vatcar.net**!
